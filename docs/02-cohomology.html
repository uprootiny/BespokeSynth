2. Cohomology & Sound

How the algebra of cochains becomes the physics of resonance.

2.1 Cochains: functions on simplices

A p-cochain is a function that assigns a real number to each p-simplex. The vector space of all p-cochains on K is written Cp(K; ℝ).

In DSP terms:
• A 0-cochain f ∈ C0 assigns a pressure to each vertex. This is the signal you hear.
• A 1-cochain g ∈ C1 assigns a flow to each edge. This is how energy propagates.
• A 2-cochain h ∈ C2 assigns a pressure to each face. This is a cavity resonance.

2.2 The coboundary operator δ

The coboundary δp: Cp → Cp+1 is the transpose of the boundary operator. It takes a function on p-simplices and produces a function on (p+1)-simplices.

δ0: gradient (vertices → edges)

(δ0f)[i,j] = f(j) - f(i)

If vertex j has pressure 5 and vertex i has pressure 3, the edge [i,j] carries a flow of 2. This is exactly the discrete gradient.

δ1: curl (edges → faces)

(δ1g)[i,j,k] = g[j,k] - g[i,k] + g[i,j]

This measures the net circulation around a triangle. If the flows around a face sum to zero, there's no curl — the flow is irrotational. If they don't, there's a vortex.

The key property: δδ = 0

Just like ∂∂ = 0, we have δ1 δ0 = 0. The curl of a gradient is zero. This is the discrete Poincaré lemma.

2.3 Cohomology: what's closed but not exact?

Definition. The p-th cohomology group is:

Hp(K; ℝ) = ker(δp) / im(δp-1)

= {p-cochains killed by δ} / {p-cochains that came from δ applied to something lower}

The dimension of Hp is the Betti number βp.

What the Betti numbers count

βpCountsSoundExample
β0 Connected components Independent fundamentals β0=1: one pitch. β0=2: two independent pitches.
β1 Independent loops Self-sustaining resonances Torus has β1=2: two loop modes that ring independently.
β2 Enclosed cavities Formant/breathing modes Tetrahedron has β2=1: one cavity that "breathes".

2.4 The Hodge Laplacian

The adjoint δ* goes backwards: it's the matrix transpose of δ. Physically, δ0* is divergence (how much flow enters a vertex), and δ1* is how much face pressure acts on an edge.

Definition. The Hodge Laplacian on p-cochains is:

Δp = δp-1(δp-1)* + (δp)*δp

For p=0 (the case we implement): Δ0 = (δ0)*δ0 = the graph Laplacian.

The Hodge theorem

The kernel of Δp (its zero eigenspace) is isomorphic to Hp(K; ℝ). The harmonic forms — eigenvectors with eigenvalue 0 — ARE the cohomology classes.

The nonzero eigenvalues are the resonant frequencies squared. The eigenvectors are the mode shapes.

2.5 From eigenvalues to frequencies

If λ1 ≤ λ2 ≤ ... are the nonzero eigenvalues of Δ0, then the resonant frequencies are:

f_k = f_base * sqrt(λ_k) / sqrt(λ_1)

where f_base is the MIDI pitch you play. The ratios √λk/√λ1 are determined purely by topology. Different complexes have different ratios — different timbres.

Example: Tetrahedron.
Graph Laplacian eigenvalues: 0, 4, 4, 4 (one zero for β0=1, three degenerate nonzero).
All nonzero eigenvalues are equal → all modes have the same frequency.
Sound: a pure tone with three-fold degeneracy (three modes at the same pitch but different shapes).
When you excite at a vertex, you hear a single pitch with a specific spatial character.
Example: Torus (7 vertices).
Eigenvalues: 0, 0.75, 0.75, 2.5, 2.5, 4.25, 4.25 (approximate).
Ratios: 1 : 1 : 1.83 : 1.83 : 2.38 : 2.38
Sound: inharmonic spectrum with paired modes (from the two-fold symmetry of the torus).
Richer, more complex timbre than the tetrahedron.

2.6 Computation in code

// Betti numbers via rank-nullity on coboundary matrices

int rankDelta0 = gaussianRank(delta0, numEdges, numVertices);
int rankDelta1 = gaussianRank(delta1, numFaces, numEdges);

beta[0] = numVertices - rankDelta0;   // connected components
beta[1] = numEdges - rankDelta0 - rankDelta1;  // loops
beta[2] = numFaces - rankDelta1;      // cavities

This is the rank-nullity theorem applied twice:

  • β0 = nullity(δ0) = N0 - rank(δ0)
  • β1 = nullity(δ1) - rank(δ0) = (N1 - rank(δ1)) - rank(δ0)
  • β2 = nullity(δ2) = N2 - rank(δ1) (since δ2 = 0 for our 2-complexes)
Computational cost: Gaussian elimination is O(N³). For N ≤ 12 vertices this takes microseconds. We compute it once when the topology changes, then synthesize per sample using only the cached eigenvalues and eigenvectors — O(modes) per sample, where modes ≤ 12.

2.7 What the math forces on the DSP

The Betti numbers aren't decorative. They constrain the DSP in precise ways:

  1. β0 = 1 means there's exactly one zero eigenvalue. The DC mode. If β0 > 1, the complex is disconnected and you get independent oscillators — the synthesis engine must handle multiple fundamentals.
  2. β1 > 0 means there are loop resonances that don't show up in the vertex Laplacian alone. The edge Laplacian Δ1 has β1 zero eigenvalues corresponding to harmonic 1-forms. These represent energy that circulates forever in loops. In the LatticeSynth, these are the ring modes.
  3. β2 > 0 means there are cavity modes. Face pressures can oscillate independently of vertex displacements. These are the "breathing" modes that create formant-like resonances. To hear them, you need the full Hodge Laplacian Δ2, not just the graph Laplacian.

Currently our CohomologySynth only uses Δ0 (the graph Laplacian). The β1 and β2 modes are counted but not synthesized. A complete implementation would compute Δ1 and Δ2 and add their eigenmodes to the output. This is the next step.
